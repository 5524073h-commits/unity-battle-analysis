# **現実とゲームにおける比較と考察**

作成日：2026年4月13日  
作成者：濵田 彪太朗

## **1\. はじめに**

本報告書は、実在する競走馬のレースデータ（JRA公式記録）を基に、ゲーム内におけるキャラクターパラメータおよびスキル効果の設計に関する論理的なアプローチを検証したものである。エンターテインメントコンテンツ、特にスポーツや対戦を題材としたゲームにおいて、リアリティとゲーム性の両立は不可欠である。本解析では、歴史的な名馬の走行データをPythonを用いて可視化し、それをゲームロジックへと昇華させるプロセスを整理した。

## **2\. 解析手法および環境**

本解析では、Google Colab環境下においてPythonによる統計処理および可視化を行った。

* **使用言語/ライブラリ:** Python , Pandas, Matplotlib, NumPy  
* **データソース:** JRA公式レース結果（2017年天皇賞（春）、2018年ジャパンカップ）、および有志によるゲーム内検証データ  
* **計算式:** ハロンタイム（200m区間秒数）から平均秒速（m/s）を算出。v \= 200 / t

## **3\. ケーススタディ1：戦術的差異の定量的比較**

### **3.1 キタサンブラック（逃げ） vs シュヴァルグラン（差し）**

2017年天皇賞（春）における両者の速度曲線を比較した結果、以下の差異が認められた。

| 馬名 | 序盤速度 (m/s) | 中盤維持力 | 終盤最大速度 (m/s) |
| :---- | :---- | :---- | :---- |
| キタサンブラック | 17.24 | 非常に高い | 17.39 |
| シュヴァルグラン | 16.81 | 温存型 | 17.70 |

分析の結果、シュヴァルグランは1200m地点まで体力を温存し、終盤区間においてキタサンブラックを0.3m/s以上上回る「末脚」を繰り出すことで、僅差の接戦を実現していることが判明した。

## **4\. ケーススタディ2：固有スキルのシミュレーション**

ゲーム内における固有スキル（速度+0.35m/s）がレース展開に与える影響をシミュレーションした。残り800m地点でスキルが発動した場合、最終的な着差は約4.8メートル（約2馬身）縮まる計算となる。これは、単なる速さの付与ではなく、ドラマチックなゲーム体験を数値的に担保するための設計根拠となる。

## **5\. ケーススタディ3：絶対的強者の定義（アーモンドアイ）**

2018年ジャパンカップにおけるアーモンドアイの世界レコードデータを解析した結果、彼女の通常速度は他の名馬のスキル発動時速度を凌駕する 18.5m/s に達していた。この外れ値とも言えるデータは、ゲームにおけるボスクラスや伝説級キャラクターのパラメータ設計において、基本ステータスのみでプレイヤーの戦略を無効化する絶望感を演出するための指標となる。

## **6\. 結論**

データ分析は、単なる事後のまとめではなく、未来の面白いゲームを作るための仕様の裏付けである。今後も論理と感性の両輪で、最高品質のコンテンツ制作に邁進したい。
